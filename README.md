# carrot-chase
Source Code for the 3D maze game
Written by M Charity

##  Notes
**Controls:**
- WASD - Move
- Mouse - Rotate Camera
- Space - Jump
- R- Reset 

**Engine**: Unity3D   

## Game Requirements (1 person):
- Has dead ends
- Has collectable items
- Has an exit leading out of the maze
